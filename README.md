# 🌦️ Know Weather Info – Weather Web Application

🔗 **Live Demo:** https://know-weatherinfo.netlify.app/

Know Weather Info is a responsive weather web application that provides real-time weather details for any location. It allows users to search for cities and view key weather parameters with a clean and intuitive interface.

---

## 🚀 Features

- 🌍 Search weather by city name  
- 🌡️ Real-time temperature updates  
- 💨 Wind speed and atmospheric details  
- ☁️ Weather conditions (Cloudy, Sunny, Rainy, etc.)  
- 📱 Fully responsive design (Mobile + Desktop)  
- ⚡ Fast and user-friendly interface  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **API:** Weather API (e.g., OpenWeatherMap or similar)  
- **Deployment:** Netlify
